# ShelfStock — backend (deployment mirror)

This repository is the **Railway deployment source** for the ShelfStock API.

📦 **The canonical repository is [jasrulete/Shelfstock](https://github.com/jasrulete/Shelfstock)** —
the full monorepo with documentation, 45 unit tests, an end-to-end suite,
Docker Compose setup, and CI running all of it on every push.

🔗 **Live demo:** [shelfstock-jer2x.vercel.app](https://shelfstock-jer2x.vercel.app/)

Express + TypeScript + PostgreSQL. Highlights: transactional checkout with
row locking, price snapshotting, row-level authorization, JWT auth with
rate limiting, and SQL-aggregate analytics.
